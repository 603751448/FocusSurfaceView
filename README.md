# FocusSurfaceView
该库主要参考IsseiAoki的SimpleCropView https://github.com/IsseiAoki/SimpleCropView </br> 
实现了在相机的预览界面指定一个区域的大小，形状和位置，只拍摄该指定区域里的图像 </br>

Supported on API Level 10 and above. </br>
